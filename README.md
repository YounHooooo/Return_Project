## 프로젝트 소개
중고거래 사이트 '번개장터'를 모티브로 만든 중고거래 웹사이트 '리턴나라'입니다.
<br/>

## 기술 스택
- Spring 5.3.24 
- MyBatis 3.4.1
- JAVA 11
- JDK 11
- lombok 1.18.24
- Oracle
- VUE3
